MontyHall

Posted on February 8th, 2008 by Ethan

I created this program to test the odds of the Monty Hall problem. The process of coding the problem actually led to me understanding why it works the way it does. I coded enough to figure out the odds of winning by sticking with the original door, and when I went to do the odds of winning by switching I realized that I didn't have to, as they're simply the opposite of the odds for not switching. It seems obvious once you've figured it out, but lots of people have trouble with this one.

CalculusClassSize

Posted on February 8th, 2008 by Ethan

Created to solve a CarTalk Puzzler. If half of a class of 1058 students qualify for Calculus, how many equally sized sections are they split up into? Clearly there can only be one answer, so this program looks for the one divisor with modulo 0. The answer is 23 sections of 23 students each.

lastFMStats

GREStudy

Posted on October 5th, 2007 by Ethan

This script is simple flash-card style GRE vocabulary study tool. IT reads a .txt file of words and definitions which I create manually from three different word of the day emails that I get every day. I would like to update this so that it can parse the emails on its own.

scheduleSort

Posted on October 5th, 2007 by Ethan

Every semester we get an Excel file that tells us when the classrooms are in use, so we know when we can work on them. It comes sorted by room number, but that's it. This script redoes the file so that there is an entry for each day for each room, and the classes on each day are sorted by start time. It makes it much easier to tell at a glance whether a room is in use.

Because this was just a quick fix, I didn't bother figuring out how to read/write .xls format from Python. The original file needs to be exported to a comma separated (.csv) document, and the output file is the same format.
