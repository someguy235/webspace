DailyMile Calendar

Grails - Groovy - JavaScript - HSQLDB - GUI - XML - DailyMile.com API
GitHub > someguy235 > dailyMileCalendar

I wanted to be able to view DailyMile entries one month at a time, but they don't offer that feature. I used the jQuery plugin FullCalendar to create my own view. This is a work in progress.

Unfortunately, the DailyMile API is kind of broken right now: The only date info returned for an event is the date that it was posted to the site, not the date that it actually took place. So lots of events show up on the wrong day, which makes this whole thing kind of pointless. I've stopped working on this until I hear something back from them.

Check it out here.
