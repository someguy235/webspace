GitHub > someguy235 > AvlCrime

NCDC Normals Access

JavaScript (jQuery) - D3.js (xCharts)

One of our scientists wanted an easy way to access Normals data, so I worked up this visualization which ended up being very popular. It's running on top of the CDO web service, and uses jQuery promises and deferred objects to manage multiple asynchronous requests.

NCDC Historical Observational Metadata Repository (HOMR) / HOMR API

Spring - Oracle - iBatis - JavaScript (jQuery) - rGraph

This is NCDC's station metadata access system, for which I am the sole developer. This is a standard Spring app using iBatis mappings into an Oracle database to serve a RESTful web service. The metadata search and visualizations are built on this service, and are presented as a single-page app. rGraph was inherited from the previous developer.

DailyMile Calendar

Grails - Groovy - JavaScript - HSQLDB - GUI - XML - DailyMile.com API
GitHub > someguy235 > dailyMileCalendar

I wanted to be able to view DailyMile entries one month at a time, but they don't offer that feature. I used the jQuery plugin FullCalendar to create my own view. This is a work in progress.

Unfortunately, the DailyMile API is kind of broken right now: The only date info returned for an event is the date that it was posted to the site, not the date that it actually took place. So lots of events show up on the wrong day, which makes this whole thing kind of pointless. I've stopped working on this until I hear something back from them.
